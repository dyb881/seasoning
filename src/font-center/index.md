# FontCenter

处理字体在部分安卓机型，上下居中时偏上

## 使用方法

```tsx
import React from 'react';
import { FontCenter } from 'seasoning';

export default () => <FontCenter style={{ border: '1px solid #000' }}>上下居中</FontCenter>;
```
