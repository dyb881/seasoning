## RouterSwitch

基于 react-router-dom 的 Switch 封装，参数一致，额外添加 routers 属性

## API

| Name    | Description  | Type                                         | default |
| ------- | ------------ | -------------------------------------------- | ------- |
| routers | 页面配置列表 | { [key: string]: React.ComponentType<any\> } |         |
